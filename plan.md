# AI算力布局地图 - 全球供应链可视化平台

## 项目概述
构建一个开源的全球AI算力供应链交互式地图，三层数据架构，每5分钟自动刷新，支持历史数据对比。

## Stage 1 — 项目架构与核心设计
**技能**: vibecoding-webapp-swarm
**子Agent任务**:
- 架构师_Agent: 设计整体技术架构、数据模型、API接口规范
- 数据模型_Agent: 设计三层数据模型、数据验证规则、来源标注规范
- UI设计_Agent: 设计交互式地图UI、图层切换、信息面板、历史对比界面

## Stage 2 — 核心地图引擎开发
**技能**: vibecoding-webapp-swarm
**子Agent任务**:
- 地图引擎_Agent: 基于Mapbox GL JS + React开发核心地图引擎、三层叠加渲染
- 数据面板_Agent: 开发侧边信息面板、企业详情弹窗、数据卡片
- 历史对比_Agent: 开发时间轴、历史快照对比、数据趋势图表

## Stage 3 — 三层数据层开发
**技能**: vibecoding-webapp-swarm
**子Agent任务**:
- 基础层_Agent: 稀土元素、光刻机、方案设计企业数据层
- 封装层_Agent: 台积电、英伟达、三星等封装工厂数据层
- 算力中心_Agent: 全球算力中心分布、用电量、供应企业数据层

## Stage 4 — 数据管道与验证系统
**技能**: vibecoding-webapp-swarm
**子Agent任务**:
- 数据管道_Agent: 每5分钟数据刷新、数据归档、历史存储
- 验证系统_Agent: 数据来源验证、交叉校验、可信度评级

## Stage 5 — 开发者体验与文档
**技能**: vibecoding-webapp-swarm
**子Agent任务**:
- 开发者友好_Agent: API文档、贡献指南、AI辅助开发配置

## 交付物
- 完整的React + TypeScript Web应用
- 数据验证与历史存档系统
- 完整的开发者文档
- 部署就绪的静态网站
