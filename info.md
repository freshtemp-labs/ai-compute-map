# AI算力布局地图 - 研究数据

## 一、全球AI算力中心分布

### 用电量与装机容量
- 国际能源署(IEA)预测：全球数据中心用电量从2025年约485.4 TWh攀升至2030年945 TWh
- 美国拥有约53.73 GW数据中心容量，占全球总数约44%
- 美国与中国合计占全球2030年前数据中心新增用电近八成
- 美国数据中心用电到2030年将比2024年增加约240 TWh
- 中国增加约175 TWh
- 数据中心带来近乎200 GW以上的容量需求

### 区域分布特征
- 美国：53.73 GW容量，约44%全球占比
- 中国：快速增长，约25-30%全球占比
- 欧洲：逐渐转向北欧、西班牙、意大利等电力充裕区域
- 数据中心向电力资源充裕的偏远地区延伸

## 二、光刻机市场数据

### 竞争格局（2024年）
- 全球集成电路光刻机出货683台
- ASML：418台（61.2%），营收235亿美元
  - EUV独家量产44台，ArFi占97.7%（129台）
  - 中国大陆收入约90亿欧元，占收入比+12pct
  - EUV均价1.9亿欧元，ArFi均价0.74亿欧元
  - 2025-2026年计划：90台EUV+600台DUV
  - 总部：荷兰费尔德霍芬
- Canon：233台（34.1%），营收16.5亿美元
  - 主攻i-line、KrF成熟制程
  - 总部：日本东京
- Nikon：32台（4.7%），营收12.5亿美元
  - 聚焦ArF、KrF等成熟DUV设备
  - 翻新机占比53%
  - 总部：日本东京

### 技术代际
- EUV（极紫外光）：ASML 100%垄断，7nm及以下制程必备
- ArFi（浸没式DUV）：ASML 97.7%份额
- ArF干式：ASML 80%，Nikon有少量
- KrF：Canon、Nikon、ASML均有
- i-line：Canon主导

## 六、数据来源与验证

### 权威数据来源
1. 国际能源署(IEA)：全球数据中心用电量预测
2. ASML年报：光刻机出货量和营收数据
3. 美国地质调查局(USGS)：稀土储量和产量
4. 各公司年报/财报：台积电、三星、英特尔等
5. 商务部/海关总署：中国稀土出口管制政策

### 交叉验证方法
- 多来源数据比对（公司年报vs行业报告）
- 地理坐标验证（公开地址信息）
- 时间戳标记（数据更新日期）
- 可信度评级（一级来源>二级来源>估算）
